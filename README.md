# Свадебное приглашение Ярослава и Марии

Готовый статический сайт для GitHub Pages. Сборка и установка программ не требуются.

Шрифт Inter и ботанический фон хранятся локально в проекте, поэтому оформление не зависит от Google Fonts или других внешних сервисов.

## Локальный запуск

Откройте `index.html` двойным щелчком или запустите локальный сервер из папки проекта:

```bash
python -m http.server 8000
```

После этого откройте `http://localhost:8000`.

## Что заменить перед публикацией

В `index.html`:

- добавьте `image_iglu.png` и `image_more.png` в папку `assets`, если этих файлов ещё нет;
- при необходимости замените изображения и подписи галереи;
- текстовые пожелания, если потребуется.

В `script.js`:

```javascript
const WEDDING_DATE = "2026-10-03T16:00:00+07:00";
const APPS_SCRIPT_URL = "https://script.google.com/macros/s/ВАШ_ID/exec";
```

Часовой пояс Новосибирска — `+07:00`.

## Замена фотографий

Положите фотографии в папку `assets` и измените пути `src` в `index.html`. Главное изображение задаётся в `styles.css` в правилах `.hero__image`, `.detail-card--image` и `.quote-photo__image`.

## Подключение анкеты к Google Таблице

1. Создайте Google Таблицу.
2. Назовите первый лист `Ответы`.
3. В первой строке создайте столбцы: `Дата`, `Имя`, `Напитки`, `Питание`, `Трансфер`, `Пожелание`.
4. Откройте `Расширения` → `Apps Script`.
5. Скопируйте в редактор содержимое файла `Code.gs`.
6. Нажмите `Развернуть` → `Новое развертывание` → `Веб-приложение`.
7. Выберите выполнение от своего имени и доступ для всех.
8. Подтвердите разрешения и скопируйте адрес, заканчивающийся на `/exec`.
9. Вставьте адрес в константу `APPS_SCRIPT_URL` в `script.js`.
10. Проверьте отправку анкеты и появление строки в таблице.

Таблицу не нужно делать публичной. В GitHub помещается только URL обработчика — паролей и ключей в проекте нет.

## Публикация на GitHub Pages

Загрузите содержимое папки в корень репозитория и отправьте изменения:

```bash
git add .
git commit -m "Обновить свадебное приглашение"
git push
```

GitHub Pages автоматически опубликует обновлённые файлы из настроенной ветки.
